This plugin give the ability to anyone how registered to your blog, to invite more people, even if the registration is closed.

== Description ==

This plugins main goal was to provide the ability to any of your registered users, to invite more people to the community.

Please notice, that this plugin totaly ignores whether the registration is enabled or disabled to your blog.

Some functions:

  * invitation sending
  * invitation cards limit per site
  * after activation, the user is logged in automatically
  * after activation, the user can be redirected to a custom URL
  * activation screen is using the same styles as the login screen (if you have customized login screen it will affect the activation screen as well)
  * Dashboard for custom settings Plugins > Invite friends settings
  * i18n support (localized messages)

== Frequently Asked Questions ==

= Does this plugin has a per user invitation card limit?  =

No. Right now it does not, but this feature will come shortly.

= On my site the registration possibility is tured off, is it possible to accept a registration card in this case? =

Yes. If you invited your friend, and he clicks on the link in the e-mail he/she can register to the blog, even is the registration is not enabled.

= Does this plugin works with WordPress 2.3 ?  =

Yes, it does :)

== Installation ==

1. Upload the ZIP files contents to the `/wp-content/plugins/invite-friends` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set customize the settings, on Invite Friends Setting dashboard, under Plugins tab.
